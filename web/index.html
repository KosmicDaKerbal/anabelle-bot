Anabelle Discord Bot

A multi-purpose discord bot made in discord.js, to help in moderation and maintaining the official discord server of r/TeensofMaharashtra.

Bot Features:

  • Captcha verification system for checking human users, which is passed on joining or on using the /captcha command.
  • /slowmode command to set custom slowmode values for channels such as 1 second, 2 seconds etc.
  • /lock and /unlock commands that help moderating the chat as per requirements.
  • "fun" commands such as /insultme, for added entertainment.

Future Update RoadMap:

  • Adding a starboard feature, which highlights a message in a channel on reaching a set number of reactions.
  • Captcha verification blacklist, which disallows any person on verifying.
  • Warning system with proper actions.
  • /flare command which allows a user to report another user to the moderators.
  • Warning system with proper actions
  • /setnick to change someone's nickname on the server
